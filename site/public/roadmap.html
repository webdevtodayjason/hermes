Shipped

[ok]Native third-party plugin — official Hermes manifest, hooks, tool, slash command, operator skill. No forks, no gateway patches.
[ok]Live nervous system — agent state from every platform; tap-to-resolve real dangerous-command approvals (with the command shown), pulse + chirp until answered.
[ok]The agent can reach the desk — a familiar_notify tool: banner, chirp, optional speech.
[ok]THE DECK — a grid of programmable buttons; each runs a script, a workflow, or a prompt, with a floating "working" pill and the result spoken + shown when it lands.
[ok]The operator's console — message scrollback, an approval detail view, and a fleet tab for background workers.
[ok]Voice + presence — the gateway's own TTS streamed to the speaker; volume, quiet hours, a nightly digest, and a morning greeting.
[ok]Untethered — runs over Wi-Fi when unplugged (auto-provisioned over USB — connect once, then cut the cord), with idle-dim battery discipline.
[ok]Pip-Boy landscape UI — six tap/swipe tabs, vitals column, banner toasts, runtime rotation config.
[ok]Browser flasher + agent-first setup — flash the factory image over WebSerial from this site, then hand your agent agent.md to finish setup.

Phases

NEXT

Landscape-Native Art

building

Today she wears portrait art adapted to the wide screen (the face above is the source art). This phase designs her for the landscape: full 320×240 scenes with the vitals column composed in. The artist spec is written.

  • Artist canvas spec — dimensions, safe zones, animation state sets, so anyone can draw her a new face  — done
  • Landscape pack export — the PNG→SD toolchain emits wide-format packs
  • Boot splash — a proper retro power-on sequence
  • Procedural CRT life — scanline drift, phosphor flicker, glitch transitions
NEXT

Sound Pack

on the speaker

Voice, volume, quiet hours, and the morning/evening digests are already in. What's left is making the little sounds good.

  • SD sound pack — real boot / alert / done chimes replacing the square-wave beeps
  • Swappable sound themes — part of the pack format, shared on the exchange below
NEXT

Community

in progress

Everything you need to build, flash, and dress your own familiar — from this site, with your agent's help.

  • Browser flasher + agent.md — shipped, see FLASH
  • Printable case — snap-fit landscape stand; CAD + Bambu 3MF land in the repo, with a print-on-demand option for the printerless
  • PACK EXCHANGE — community face & sound packs: browse, download, submit your own. The pack format already exists; the exchange makes her a canvas
  • Public repo + demo mode — runs standalone without a gateway, for the bench
LATER

Familiar Everywhere

later

The transport is already networked, so the "device" no longer has to be an ESP32. The same plugin can drive a second surface in your pocket.

  • Phone app — the same tabs, ticker, approvals, and deck, from anywhere
  • Push notifications — approvals reach you away from the desk
  • Multi-familiar — several devices, one gateway: desk, workshop, rack room
