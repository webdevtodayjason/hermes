THE PLUGIN

Everything on the host side is a standard third-party Hermes plugin — manifest, hook registration, a registered tool, a slash command. No gateway patches, no forks. It lives inside your always-on gateway process: no daemon, no polling.

Install

hermes plugins install webdevtodayjason/hermes/plugin --enable
~/.hermes/hermes-agent/venv/bin/pip install pyserial
hermes gateway restart

Plug the familiar into any USB port — she's found automatically and survives unplug/replug. /familiar in any Hermes chat shows link status; /familiar say hello is the ten-second voice test.

note  The repo goes public with the first release — until then this command works for collaborators only.

Voice uses your gateway's own TTS config (ElevenLabs, OpenAI, Edge — whatever you run); audio streams over your LAN to her speaker. Configuration lives in ~/.hermes/familiar_actions.json — action buttons, voice, quiet options, all documented in the repo.
